# Project Skills Directory

This directory contains specialized AI agent skills tailored for the `next-production-template` project.

## How to use a skill
You can activate a skill by name during a session if it's registered, or Gemini will automatically reference these instructions if they are mentioned in `GEMINI.md`.

## Directory Structure
- Each folder represents a skill.
- Each skill folder must contain a `SKILL.md` file.
- `SKILL.md` should define the specialized expertise, tools, and workflows for that specific domain.

## Available Local Skills
- **shared-logic-expert**: Expert in managing Zod schemas and shared types across the monorepo.
